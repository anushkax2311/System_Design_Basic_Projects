# Java Low Level Design (LLD) Projects 🚀

This repository contains multiple **Java-based Low Level Design (LLD) projects** built to practice **Object-Oriented Programming (OOP)**, **Design Principles**, and **System Design concepts**.
Each project demonstrates real-world system modeling using clean architecture and modular design.

---

## 📂 Projects Included

### 1. 🎟️ Discount Coupon System

A coupon management system that supports:

* Percentage & flat discounts
* Coupon validation
* Expiry handling
* User eligibility checks
* Discount calculation logic

**Concepts Used**

* OOP
* Strategy Design Pattern
* Encapsulation
* Polymorphism

---

* Collections Framework

# ▶️ How to Run

### Clone the repository

```bash
git clone <your-repository-url>
```

### Open in IDE

Import the project into:

* IntelliJ IDEA
* Eclipse

### Compile & Run

Run the main class of the respective project.

Example:

```bash
javac Main.java
java Main
```
